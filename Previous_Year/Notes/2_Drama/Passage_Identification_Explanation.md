## 🔎 HOW TO RECOGNIZE PASSAGES (General Trick)

When you see the extract, ask yourself:

1. Who is speaking?
2. Is it emotional / comic / philosophical / political?
3. Any famous line or keyword?
4. Is it early, middle, or climax?

Look for:

* Tone (angry? guilty? romantic? comic?)
* Key words (soul, revenge, honour, wit, salvation, etc.)
* Relationship between characters

# 📘 DR. FAUSTUS – Important Passages

### Most Important Speeches:

1️⃣ “Was this the face that launch’d a thousand ships?”
→ Speaker: Faustus
→ Situation: He sees Helen
→ Theme: Illusion, desire, damnation
→ Trick: If you see romantic admiration + classical reference → It’s Helen scene.

2️⃣ “O, I’ll leap up to my God! Who pulls me down?”
→ Speaker: Faustus
→ Final scene before death
→ Theme: Fear, repentance, despair
→ Trick: If you see fear of hell, clock striking, time ending → Last act.

3️⃣ “Why, this is hell, nor am I out of it.”
→ Speaker: Mephistopheles
→ Theme: Hell is spiritual suffering
→ Trick: Philosophical tone about damnation = Mephistopheles.

# 📘 HAMLET – Important Passages

1️⃣ “To be or not to be”
→ Hamlet
→ Deep thinking, life/death
→ If passage sounds philosophical → Hamlet soliloquy.

2️⃣ “O that this too too solid flesh would melt”
→ Early grief speech
→ Anger at mother

3️⃣ Graveyard scene (Yorick skull)
→ Reflection on death
→ If skull mentioned → Act 5

4️⃣ Play-within-play scene
→ Acting, performance, guilt
→ If Claudius reacts → Mousetrap scene.

Trick: Hamlet passages are usually long and thoughtful.

# 📘 THE WAY OF THE WORLD (Comedy)

Tone: Witty, sarcastic, clever dialogue.

Most Important Extracts:

1️⃣ Proviso Scene (Mirabell & Millamant)
→ Discussion about marriage terms
→ If funny but serious about independence → That scene.

2️⃣ Lady Wishfort scenes
→ Dramatic, foolish pride
→ If exaggerated emotional language → Lady Wishfort.

Trick: If language feels modern and witty → It’s Congreve.

# 📘 SAINT JOAN (George Bernard Shaw)

Tone: Strong, political, argumentative.

Important Extracts:

1️⃣ Trial Scene
→ Joan defending herself
→ Religious authority conflict
→ If church vs individual debate → Trial.

2️⃣ Joan hearing voices
→ Spiritual conviction
→ If confidence + divine mission → Early acts.

Trick: Shaw uses rational debate style, not poetic language.

━━━━━━━━━━━━━━━━━━

# 📘 THE ZOO STORY (Edward Albee)

Tone: Psychological, tense, modern.

Important Extracts:

1️⃣ “I’ve been to the zoo.”
→ Opening line
→ Jerry starts conversation.

2️⃣ Dog story
→ Symbolic speech about loneliness
→ If long strange story → Jerry.

3️⃣ Final fight scene
→ Knife, park bench
→ Climax tragedy.

Trick: Only 2 main characters (Jerry & Peter). Easy to identify.

# 🎯 MASTER EXAM TIPS

✔ Read beginning and ending lines carefully.

✔ Identify speaker first — that gives 50% answer.

✔ Write answer in this format:

1. Speaker
2. Context
3. Meaning
4. Themes
5. Significance in play

✔ Remember climax scenes — examiners LOVE climax.

✔ Focus on:

* Final scenes (Dr Faustus, Hamlet)
* Trial scene (St Joan)
* Proviso scene (Way of the World)
* Bench fight (Zoo Story)
