## ✨ Overview

This repository is a **structured study hub** for M.A. English Literature, containing:

* 📝 Markdown notes (well-organized & exam-focused)
* ✍️ Handwritten notes and scanned materials
* 📄 Previous year question papers
* 🔗 Important academic links & references
* 🎯 Revision sheets and quick summaries
